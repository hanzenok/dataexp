    Class LeftNavCtrl

    Class defined in: app/js/controllers/LeftNavCtrl.js:7

    A controller that serves all the left panel in the index.html view.
    It uses three custom angular services:

    • SourcesService: deals with getting/sendig/adding/deleting sources from/to/to/from the backend
    • StoresService: deals with getting stores from the backend
    • FieldsService: deals with getting fields from the backend
