    Class DCChartsService

    Class defined in: app/js/services/DCChartsService.js:7

    A service that wraps a DC.js charting library and offers two public methods:

    • load(): loads the data (json array) that should be visualysed
    • getChart(): returns the chart filled with the data from the load() function. The chart can be then traced by calling render() function on it.

    String color_gen( value )
    app/js/services/DCChartsService.js:441

    A local method used by timeline to color the correlated fields. It is using an algorithm to generate a color (hexa format, without alpha) from a [0..1] value.

    Parameters:
    • value <Float>

      A [0..1] value from which the color should be generated


    Returns: <String>

    A color in hexa format

    Example
    console.log(color_gen.call(this, 0.7338)); //55aa2a
console.log(color_gen.call(this, 0.9321)); //341a69
