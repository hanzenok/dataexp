    Class LoaderCtrl

    Class defined in: app/js/controllers/LoaderCtrl.js:7

    A controller that serves loader zone in the footer of index.html view.
    It uses three custom angular services:

    • TimeseriesService: deals with getting the timeseries, tsproc config and timeseries statistics from the back-end
    • DCChartService: used to load data to the DC.js charting library
    • CanvasChartService: used to load data to the Canvas.js charting library
