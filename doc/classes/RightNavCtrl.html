    options <Json>
    app/js/controllers/RightNavCtrl.js:55

    A local scope variable that holds some tsproc configurations.
    Transformation configs:

    • transform_type: the type of transformation to apply in order to synchronise multiple timeseries. Possible values are 'interp' (interpolation) and 'inters' (intersection)
    • interp_type: interpolation method. Possible values are: 'linear', 'cubic', 'lanczos', 'nearest' (nearest neighbor)

    Reduction configs:

    • reduc_type: the reduction method to combine $scope.options.reduc_size datapoints into one. Possible values are: 'skip', 'sum', 'avg', 'max', 'min'
    • reduc_size: the number of datapoints to be reduced into one the $scope.options.reduc_type method
    • target_field: a target field used for 'max' and 'min' methods

    Date configs:

    • from_date: a date from which to cut the dates
    • to_date: a date till withc to cut the dates

    Correlation configs:

    • count_negative: wheather or not to count the negative correlation as a correaltion
    • max_coef: wheather or not to use the maximum correlation value as a criteria to consider the set of datapoints as correlated. If max_coef is false, use max number of datapoints as a criteria

    Quantification config:

    • tsfield_quantum: the quantum of the timestamp attribute. Possible values are: 'none', 'day', 'month', 'year'
    size_status <String>
    app/js/controllers/RightNavCtrl.js:27

    A global scope variable that holds the state of the timeseries size. It is equal to 'overflow' when the size is bigger than $rootScope.max_size or 'normal' otherwise.

    stats <Json>
    app/js/controllers/RightNavCtrl.js:38

    A local scope variable that holds three timeseries statistics:

    • homogen: wheater or not one or multiple timeseries where homogeneous before the merge
    • size: a size of the final fused timeseries
    • per_day: an average number of datapoints per day in the timeseries
