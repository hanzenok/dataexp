deleteSource( req, res )
server/rest/DeleteSource.js:16

A method that removes a source configuration from the ./server/config/sources.json file. The source to delete is specified by it's name in the req object.
If the source to delete is a file (JSON or CSV), then it would be deleted from the directory ./server/datasets/.
An empty array is returned by the res object.
