    Class DialogCtrl

    Class defined in: app/js/controllers/DialogCtrl.js:7

    A controller that serves the two templates for dialog windows: AddSource.html and SaveFormat.html.
    It uses two custom angular services:

    • SourcesService: deals with getting/sendig/adding/deleting sources from/to/to/from the backend
    • CSV2JSONService: used for csv type sources

    saveFormat( )
    app/js/controllers/DialogCtrl.js:330

    A local scope method that serves SaveFormat.html dialog and saves the the timestamp format specified by user.
    It is done by calling the method hide() of $mdDialog service and passing it the format. This way the format can be grabbed in the .then() method in the controller LoaderCtrl.
